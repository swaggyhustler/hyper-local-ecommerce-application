
## Add Shops
### Mehtod `POST` 
### URL `http://localhost:5000/api/v1/add/shop`

### Required Details
Form Data
- Image of the shop (1), `name='image' type='file'`
- Name of the shop, `name='shopName' type='text'`
- Coordinates, `name='coordinates' type='text'`
- owner_id, `name='owner_id' type='text'`
